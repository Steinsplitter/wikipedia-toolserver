some undocumented stuff:

For all requests, you can also use GET requests.

For the histogram, you can set the paramters:
--hist = bin sizes in hours
--h = kernel smoothing parameter h (if negative, no smoothing is performed)

For the history, you can set the parameters:
-- hist_day = how many days back the history should go (-1 == all history)
You can also show one month only, e.g. Monat April
